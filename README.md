# unidata2
Aprimoramento do arquivo "uni_data_mesma_regi2.dbf" gerado pelo programa "duplicados.exe" (https://github.com/Regional-Entorno-Sul/duplicados), contém, às vezes, municípios de notificação que nao correspondem ao que está registrado no SIVEP Gripe.  
Este utiltário cria então o arquivo "uni_data_mesma_regi3.dbf" com os campos "mun_left" e "mun_right" mostrando o real município de notificação dos casos listados.
